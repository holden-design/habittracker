# Personal Systems

A visual desktop productivity application built with Electron, React, and TypeScript.

## Getting Started

### Installation

```bash
npm install
```

### Development

Run the app in development mode:

```bash
npm run dev
```

This will start both the React development server and the Electron app.

### Building

Build the application:

```bash
npm run build
```

Create a distribution package:

```bash
npm run dist
```

## Project Structure

- `src/main/` - Electron main process
- `src/renderer/` - React components (renderer process)
- `public/` - Static assets
- `dist/` - Compiled main process (generated)
- `build/` - Built React app (generated)

## Features

- Cross-platform desktop app (Windows, macOS, Linux)
- Modern React with TypeScript
- Electron for native desktop capabilities
- Hot reload development experience
